### 一、用户模块

#### 1.登录

/user/login

> request

    email，password

    
> response

fail

    {
        "status":1,
        "msg":"密码错误"
    }

success

    {
        "status":0,
        "data":{
                    "id": 12,
                    "nickname": "aaa",
                    "email": "aaa@163.com",
                    "sex": null,
                    "role": 0,
                    "createTime": 1479048325000,
                    "updateTime": 1479048325000
                   }
    }
    
#### 2.注册

/user/register

> request

    email，nickname，password，sex

    
> response

fail

    {
        "status":1,
        "msg":"邮箱已被注册"
    }

success

    {
        "status":0,
        "msg":"注册成功"

    }

#### 3.忘记密码

/user/forgetpw

> request

    email

    
> response

fail

    {
        "status":1,
        "msg":"邮箱（或验证码）错误"
    }

success

    {
        "status":0,
        "msg":"修改密码成功"
    }
